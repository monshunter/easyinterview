# 讨论文档模板

## 1 Agent 分析模板

```markdown
# ${Subject} 分析

> **Agent**: Claude
> **日期**: YYYY-MM-DD

## 1 问题理解

对问题的理解和分析背景。

## 2 分析过程

### 2.1 方案一

描述和评估。

### 2.2 方案二

描述和评估。

## 3 建议方案

推荐的方案及理由。

## 4 风险与注意事项

- 风险点 1
- 注意事项 1
```

## 2 汇总文档模板

```markdown
# ${Subject} 讨论汇总

> **日期**: YYYY-MM-DD

## 1 背景

描述讨论的背景和目标。

## 2 各方观点

### 2.1 Claude 观点

来源：[详细分析](./xxx-by-claude.md)

- 要点 1
- 要点 2

### 2.2 Gemini 观点

来源：[详细分析](./xxx-by-gemini.md)

- 要点 1
- 要点 2

## 3 决策

**采纳方案**：简要描述最终方案

**决策依据**：
- 原因 1
- 原因 2

**来源**：主要采纳自 [Agent 名称] 的建议
```
