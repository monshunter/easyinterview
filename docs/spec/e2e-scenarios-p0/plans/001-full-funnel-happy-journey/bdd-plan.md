## 2 Failure and Recovery

- Empty optional arrays must never become SQL `NULL`.
- Completion must not address removed columns.
- Report persistence must use PostgreSQL array encoding.
- A retrying report may continue from `generating` without re-reservation failure.

## 3 Privacy

Evidence records only redacted observations, resource/status summaries, and screenshot paths. It excludes secrets, codes, cookies, full prompt/response bodies, and private input prose.
