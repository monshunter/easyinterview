## E2E.P0.007 Disabled phone mode

Given a running text conversation and disabled voice profiles, when the user views/operates the phone icon, loads stale phone params, or calls the voice endpoint, then the icon is unavailable, the UI remains text-only, and the backend returns typed unsupported with zero provider/persistence side effects.
