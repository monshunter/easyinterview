# Mock Interview 当前面试规划目标模块

> **版本**: 1.5
> **状态**: active
> **更新日期**: 2026-05-01

## 1 文档目的

本文档定义当前静态 UI 中 `模拟面试` 一级模块的目标结构。该页面不再定义为“当前岗位”，而是用户开始一场模拟面试前的上下文确认与规划页面。

## 2 模块职责

当前面试规划负责：

- 明确这场模拟面试绑定的目标岗位 / JD。
- 明确这场模拟面试使用的简历版本。
- 明确当前处于哪一轮真实面试节点。
- 允许用户切换已有面试规划或新建规划。
- 允许用户在弹窗中更换绑定简历。
- 展示 JD 拆解、公司情报轻量卡片、我的准备和最近模拟面试历史。
- 允许从公司情报轻量卡片打开 `company_intel` 详情页，并返回面试前确认。
- 提供唯一主入口：`立即面试`。

当前面试规划不负责：

- 作为“当前岗位”一级模块存在。
- 管理多岗位资产的全部生命周期。
- 管理简历版本资产。
- 展示练习模式卡片。
- 提供热身、反问、单题深钻或针对性复练。
- 承载无上下文的面试报告入口。

## 4 关键交互

### 4.1 进入当前面试规划

```text
入口:
├─ Home 最近模拟面试卡片
├─ Home JD 解析确认
├─ Job Picks 推荐 JD -> parse JD 解析确认
├─ Report 的复练当前轮
└─ Report 的进入下一轮

进入:
  -> Mock Interview Plan(jobId, resumeVersionId, roundId)
```

无论从哪个入口进入，页面都必须明确显示 `JD + 简历 + 面试轮次`。

### 4.2 切换或新建规划

```text
用户不想继续当前规划
  -> 点击切换规划
     ├─ 选择已有 TargetJob / ResumeVersion / Round 组合
     └─ 更新当前页面上下文
  -> 或点击新建规划
     └─ 回到首页导入新的 JD
```

这解决用户从一级 `模拟面试` 进入后，不想继续最近规划时无路可走的问题。

### 4.3 更换简历

```text
绑定简历卡片
  -> 更换
  -> Resume Picker Modal
     ├─ 展示已有简历名称
     ├─ 展示来源 / 主版本 / 匹配信息
     ├─ 选择一个 ResumeVersion
     └─ 确认绑定到当前规划
```

更换简历不应直接跳转到简历模块。用户要做的是为当前模拟面试选择输入，而不是离开流程管理简历资产。

### 4.4 立即面试

```text
Mock Interview Plan
  -> 立即面试
  -> Interview Session
     ├─ 文本面试
     └─ 语音面试
```

面试形式可在面试页选择或切换。规划页不展示模式卡片，也不让用户选择专项练习。

### 4.5 查看模拟面试历史

```text
右侧模拟面试历史
  -> 点击某一次会话
  -> Report Dashboard(sessionId)
```

这里展示的是完整模拟面试会话历史，标题不能再使用旧版“岗位定制核心面、单题深钻、英文热身、反问专练”等专项流程。

## 6 信息层级

```text
最高优先级:
├─ 这场模拟面试绑定哪份 JD
├─ 这场模拟面试绑定哪份简历
├─ 当前处在哪一轮真实面试节点
├─ 如果不继续当前规划，如何切换或新建
└─ 立即面试

次级信息:
├─ JD 拆解
├─ 公司情报轻量卡片 / 详情
├─ 直接命中
├─ 风险提示
└─ 模拟面试历史

低频信息:
├─ JD 原文
├─ 数据来源
└─ 更新时间
```

## 7 与被移除模块的边界

| 被移除模块或流程 | 当前面试规划内是否保留相关能力 | 边界 |
|------------------|-------------------------------|------|
| 成长 | 否 | 不展示长期成长中心 |
| 多轮计划 | 否 | 只展示当前目标岗位的面试轮次节点，不维护独立计划系统 |
| 经历库 | 否 | 面试可读取简历和画像，但不要求用户维护经历库 |
| 追问树 | 否 | 追问在面试会话内发生，不作为入口 |
| 针对性复练 | 否 | 报告可发起复练当前轮，但不是单题 Drill |
| 练习模式选择 | 否 | 只有 `立即面试` 主动作 |

## 8 后续实现输入

1. 顶部导航文案为 `模拟面试`。
2. 页面主语使用 `当前面试规划`，不得再用 `当前岗位` 表示一级模块。
3. 主 CTA 文案为 `立即面试`。
4. 顶部线条节点展示真实面试轮次，不展示模拟面试内部题目。
5. 绑定简历的 `更换` 动作必须打开选择弹窗。
6. 右侧历史是完整模拟面试会话历史，不是专项练习历史。
7. 报告从历史会话或面试结束后进入，报告页必须带 session 上下文。
8. `company_intel` 是从规划页打开的岗位情报详情，不进入顶部导航。
